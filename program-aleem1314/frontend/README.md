# Pulse App - Frontend

A modern, minimalist Web3 social application built with React, Vite, TailwindCSS, and Solana Wallet Adapter.
Users can connect their wallet, create a profile, upload profile images to Pinata (IPFS), and create posts stored on the Solana blockchain.

## Environment Setup

Create a `.env` file in the root of your frontend:

``` ini
VITE_PINATA_JWT=YOUR_PINATA_JWT_HERE
```

## Running the Frontend

Install dependencies:

``` sh
npm install
```

Start the dev server:

``` sh
npm run dev
```

## Build for production

``` sh
npm run build
```
Preview locally:

``` sh
npm run preview
```
